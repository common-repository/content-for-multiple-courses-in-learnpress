== Description ==  

This plugin allows you to add lessons and quizzes in multiple courses. It extends the original <a href="http://thimpress.com/learnpress">LearnPress</a> plugin, which allows to add lessons and quizzes to just one course exclusively.

== Installation ==  

**From your WordPress dashboard**  
1. Visit 'Plugin > Add new'.  
2. Search for 'Content for multiple Courses in LearnPress'.  
3. Activate LearnPress from your plugins page.  

**From WordPress.org**  
1. Search, select and download 'Content for multiple Courses in LearnPress'.  
2. Activate the plugin through the 'Plugins' menu in WordPress dashboard.
